Legal disclaimers, sources of information, and security

All information in this tool is derived from public sources. The sources are listed below. The information included may be incorrect because it is either from released tech data from F-16A/B variants, outdated TOs approved for public release, or public sources with little to no credibility.

All data in this tool is stored in a single JSON file. Format is under JavaScript object syntax and is unencrypted.

There is no DoD instruction, DAFI, or AFI that prohibits note-taking. This is note-taking, with ease-of-use tools and better navigation. That being said, this information is not easily accessible.

SECURITY

This system uses a built in API called Web Crypto AES (Advanced Encryption Standard). It is considered secure but technically not required for use of this platform. This program does not connect to the internet, or have any access to the internet. It relys on the phone security. The reason Web Crypto API is used is for potential controlled distribution of this contents of this program.

The data in the file contains all of the information (listed below), except for fuel load calculator, oil consumption calculator, and torque converter.

The encrypted file is embedded with user identifiable information. The creator of the encrypted file is responsible for this information being accurate. This is used in case of distribution of the encrypted file, the person responsible can be held accountable.

No data is ever stored as readable information. JSON (JavaScript Object Notation) is what is saved behind the encrypted file. The data is saved between uses using IndexedDB. This is open and readable in a normal website but in this case it's an encrypted string. The decryption key is only saved as a local variable and gets wiped every time the app closes.

Embedded within the file is identifiable information that, if released, can identify who released it. Also an expiration date that renders the file useless after a set date (like when a contract ends)

Torque converter

This does not change the torque applied to the fastener. When using an inline extension, torque wrench settings must be adjusted to prevent over- or under-tightening of a fastener, using standard torque compensation formulas. Always verify torque procedures with applicable Technical Orders. The formula and explantion are below.

The torque equation is as follows:

Tapplied = Tsetting × ( (L + E × cos(A)) / L )

Where:
Tsetting = Setting on torque wrench
Tapplied = Desired torque at fastener
L = Length of torque wrench (center of handle to drive)
E = Length of extension (center of drive to fastener)
A = Angle of extension (0deg being directly outward)

Units are irrelevant because this formula scales Tapplied
